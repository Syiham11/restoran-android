# restoran-android
aplikasi menu restoran dengan menggunakan android

features
1.menambah tempat (meja)
2.mengirimkan pesanan ke dapur
3.mengirimkan total harga dan rincian pesanan ke kasir
4.menambahkan menu (makanan dan minuman)
5.menu deposit (untuk deposit uang, sehingga pelanggan tidak perlu membayar dengan uang, namun bisa menggunakan deposit uang yang dimiliki)
6. dll
